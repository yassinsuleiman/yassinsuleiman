## 🔥 Featured Project

### 🚀 [Gatus ECS Deployment Project](https://github.com/yassinsuleiman/gatus-ecs)

A production-ready infrastructure-as-code deployment of **Gatus** on **AWS ECS (Fargate)** using **Terraform** and **GitHub Actions (OIDC)**.

**Tech:** `AWS` `ECS Fargate` `Terraform` `Docker` `GitHub Actions` `Trivy` `Checkov` `ACM` `Route 53` `ALB`

**What it does**
- ✅ Runs containers on **ECS Fargate** (serverless compute)
- ✅ **Multi-AZ** design for higher availability
- ✅ **Private subnets** for tasks (no public IPs)
- ✅ **ALB + HTTPS** using **ACM** with a custom domain via **Route 53**
- ✅ CI/CD workflows: **build → scan → plan → apply** (+ post-deploy **/health** verification)
- ✅ Security gates: **Trivy** (container scan) + **Checkov** (IaC scan)
- ✅ Terraform state: **encrypted S3** + **locking** (S3 locking)

**Key metrics**
- ⚡ ~6.8 min automated Terraform deploy (incl. scans + health verification)
- 📦 3.5 GB → 75 MB (~98%) image size reduction using multi-stage Docker builds
- 🔐 OIDC → AWS IAM role (no long-lived AWS keys)
